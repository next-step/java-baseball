# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

1. 상대방(컴퓨터) 값 선택
	- 임의의 수 3개 발생
	- 중복 불가
	- 발생 된 수 저장

2. 입력한 숫자에 대한 결과를 출력	
	- 입력 받은 값 유효성 체크
		- 수 만 입력했는지 체크
		- 수 범위(3자리 까지) 체크
		- 숫자 중복
		- 입력 받은 값 저장
		- 유효성 체크 실패 시 유형 별 유효성 관련된 문구 노출(UI)
	- 비교
		- 스트라이크 체크
		- 볼 체크
	- 결과
		- 결과 출력 (UI)
			- 스트라이크 or 볼
			- 낫싱
	- 게임 진행 여부 결정
		- 3 스트라이크 성공
			- 게임을 다시 시작 하거나 완전히 종료 문의(UI)
		- 그외
			- 다시 입력 대기 상태(UI)
	
3. 게임을 다시 시작 하거나 완전히 종료
	- 다시 시작하기
	- 완전히 게임 종료
