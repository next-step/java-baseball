# 숫자 야구 게임

## 기능 목록
* 게임 시작 기능
* 게임 종료 기능
* 랜덤 숫자 생성 기능
* 스트라이크 검증 기능
* 볼 검증 기능
* 게임 승리 검증 기능

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
