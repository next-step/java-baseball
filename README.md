## 기능 목록
1. 사용자의 숫자 야구 입력값 검증
2. 에러 메시지 출력
3. 스트라이크/볼 확인
4. 랜덤 숫자 생성
5. 틀렸는지 맞췄는지 여부 체크
6. 맞췄을 경우 계속 할건지 여부 체크
7. ui 출력
