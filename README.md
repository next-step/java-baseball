# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
 * 사용자 화면 
        + 사용자 숫자 입력
        + 사용자 게임 재시작/종료 입력
        + 결과 출력
 * 게임 기능
        + 3자리 난수 생성
        + input 값 체크(유효성 검사)
        + Strike 갯수 확인
        + ball 갯수 확인
        + 게임 종료 후 재시작/종료
