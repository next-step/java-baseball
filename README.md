# 야구 게임 기능 목록

- 게임을 실행하는 UI 기능
    - 게임을 생성해서 실행
    - 게임이 종료되면 새로 시작, 종료할 수 있는 기능
        - 1 또는 2가 입력되는지 예외 처리
- 게임 핵심 로직
    - 문제 초기화하는 기능
        - 0~9 숫자 중의 서로 다른 3개의 숫자를 선택하는 기능
    - 입력받은 답 체크하는 기능
        - 입력받은 숫자가 답과 동일하면 게임 종료
        - 볼이 몇개인지 확인하는 기능
        - 스트라이크가 몇개인지 확인하는 기능
        - 낫싱인지 확인하는 기능
    - 답 입력받는 기능
        - 입력받은 문자가 세개의 숫자인지 예외처리
