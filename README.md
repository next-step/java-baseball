# 숫자 야구 게임

## 기능 정의
* 1~9의 숫자중 중복되지 않게 랜덤숫자를 생성
* 유저의 숫자를 입력
* 두개의 숫자를 비교 기능
* 출력 기능
* 종료 여부 기능

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
