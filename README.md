# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 기능목록정리

- 기능목록 정리
- 학습테스트

- 숫자 입력기능
  - 서로 다른 3개의 숫자를 입력받는다
- 컴퓨터 수 랜덤으로 생성 기능
  - 서로 다른 3개의 숫자를 생성한다
- 숫자 검증 기능
  - 같은 수가 같은 자리에 있으면 스트라이크
  - 다른 자리에 있으면 볼
  - 같은수가 전혀 없으면 포볼 또는 낫싱
- 결과 출력 기능
