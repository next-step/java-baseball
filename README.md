야구게임 기능 목록
=============
* 세 자리 난수 생성
  * 서로 다른 3개의 숫자로 구성
    

* 입력 메시지 추가
  * 숫자 입력
  

* 입력받은 수 확인
  * 정수 인지
  * 3자리 인지
  * 중복이 없는지
  
    

* 입력받은 숫자에 대한 결과 확인
  * 숫자와 위치가 모두 일치 -> 성공
  * 숫자와 위치가 일치 -> 스트라이크
  * 숫자가 일치하지만 자리가 다르면 -> 볼
  * 숫자와 자리가 모두 다르면 -> 낫싱
    

* 출력 메시지
  * 결과 출력 메시지
  * 성공시 재시작 여부 메시지
    

* 재시작 여부 확인
  * 1 입력 시 재시작
  * 2 입력 시 종료


* 에러 메시지
  * 올바른 세 자리 숫자 입력 하지 않을 경우 
  * 게임 재시작 여부 확인시 유효하지 않은 문자 입력
