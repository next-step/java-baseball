# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
* 서로 다른 임의의 3개의 숫자 생성
* 사용자에게 3개의 수 입력 받기 
* 입력값에 대한 힌트 출력 
* 정답이 아닐 경우 사용자에게 숫자입력을 재요청
* 정답일 경우 게임 다시시작 및 완전히 종료
* 사용자 입력값 예외처리
