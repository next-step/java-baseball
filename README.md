# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록 (GameHost)
### 플레이어
* 서로 다른 3자리 숫자 입력

### 컴퓨터
* 서로 다른 3자리 랜덤 숫자 생성
* 게임 진행, 모두 맞출 때까지 아래 반복 
    * 플레이어가 입력한 수와 생성된 수 비교
    * 비교 결과 출력 
* 게임 종료 or 다시 시작
