# 기능 정리
1. 1에서9까지 숫자를 랜덤하게 생성
   * 숫자에 중복값이 없는지 확인
   * 숫자 자리수가 3자리가 될때까지 루핑

2. 사용자 입력한값을 판단함.
   * 스트라이크,볼을 체크한후에 결과 카운트를 dto 에 저장
   
3. 입력 값에 대한 유효성 체크
   * 숫자만 인지 확인
   * 3자리가 넘는지 확인
  
4. 게임이 종료 되었는지 확인
   * 3스트라이크일 경우 재시작 여부 확인
   * 1:재시작 ,2:종료 그 외에는 재시작 여부 확인 
     올바른 값이 들어올때까지 루핑
