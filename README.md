# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록

* 임의 숫자 선정 기능
    * 1~9 랜덤 숫자 뽑기 func
    * 같은수 validate func

* input handling
    * 사용자 input validate func

* 게임 비교 로직
    * 각 자리값 비교 func
    * 다른 자리값 비교 func
    
* 기능 연동 및 서비스 메인    
    * 결과 return 및 재사용
