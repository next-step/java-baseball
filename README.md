# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 구현 기능 목록
* 랜덤 숫자 생성
** 사용자가 입력한 숫자 범위와 추출할 갯수를 기반으로 랜덤 숫자 배열을 생성한다.
* 점수 계산 로직
** Strike 확인
** Ball 확인
* UI
** 사용자가 입력한 숫자를 배열로 변경한다.
** 게임을 위한 UI를 만든다.
