# 숫자 야구 게임
#기능구현 List
##Number
    - 숫자 생성
    - 0~9 범위의 숫자를 가진다. (검증)
##Numbers
    - 3개의 숫자로 구성
    - 3개의 숫자가 같은 수가 있는지 검증
##Hitter(숫자문제를 내는 컴퓨터)
    - 중복되지 않는 랜덤 숫자 3개 문제를 낸다.
##Picther(숫자를 맞추는 사람)
    - 숫자 3개를 임의로 입력한다.
##referee
    -컴퓨터가 낸 문제를 숙지한다.
    -사용자가 제출한 숫자를 문제와 비교해서 판단한다.
##RandomNumber
    - RandomNumber(범위 0-9)를 생성한다.

##baseBallGame
    - Game을 생성한다.
    - 다시 Game 재시작할지 정한다.
##InputView
    - 사용자가 입력하는 View
##OoutputView
    - 사용자에게 보여주는 View
