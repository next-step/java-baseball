# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

# 기능 명세
- [x] 1 ~ 9 사이의 서로 다른 숫자 3개를 선별하는 기능
- [ ] 숫자 3자리를 입력받는 기능
- [ ] 스트라이크 판별 기능
- [ ] 볼 판별 기능
- [ ] 포볼 또는 낫싱 판별 기능
- [ ] 게임 결과 출력 기능
- [ ] 숫자 맞출때까지 반복하는 기능 
- [ ] 게임을 종료한 후 게임을 다시 시작하거나 완전히 종료하는 기능
