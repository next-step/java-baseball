# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록

BaseBallGameService 
* 입출력에 따른 기능 게임 스타트
* 게임 완료 여부 체크
* 게임 메시지 전달
* 게임의 진행되는 TURN 기능
* 컴퓨터에게 임의의 숫자 3개 를 전달

BaseBallGameValidator 
* 유저의 입력값 SIZE 체크
* 유저의 입력값 숫자인지 체크
* 게임 시작시 1,2 체크
