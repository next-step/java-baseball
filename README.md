# 숫자 야구 게임

## 구현 기능 목록
- 해당 자릿수에 값을 추출하는 메소드
- 특정한 값이 랜덤값 3자리에 포함여부와 자릿수와 같은지 여부
- 최종 결과를 반환하는 메소드
- 중복되지 않은 랜덤 숫자 3자리를 반환하는 메소드
- 사용자 UI 

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
