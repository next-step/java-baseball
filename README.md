# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
1. 상대(컴퓨터) 인터페이스 구현
    - 정답 생성 : 임의의 세자리 숫자 생성
    - 사용자 입력 검사 : 사용자 입력 숫자를 받아 정답과 비교
        - input : 예상 값 (사용자 입력값)
        - output : 검사 결과 (int 배열) 
            - index 0 : Strike 갯수
            - index 1 : Ball 갯수
2. UI 구현
    - 메뉴 : 게임 시작 종료 (1: 시작, 2: 종료)
    - 정답 제출 -> 결과 출력
        - 성공 시 메뉴로 돌아감
        - 실패 시 정답 제출 상태로 돌아감
