# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 구현 목록
* 랜덤 숫자 생성
    - 1~9, 자리는 3자리, 중복 불가, 정렬 필요 없음.
* 사용자 입력 기능
    - 숫자만 입력 가능, 3자리 입력 제한
* 입력된 숫자와 랜덤 숫자 검증
    - 자리와 숫자가 맞으면 스트라이크
    - 숫자만 맞으면 볼
    - 아무것도 아니면 낫씽
    - 스트라이크 3개면 모두 맞힘
* 종료 혹은 계속
