# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

##구현 기능
1. 준비
    * 파일명 : BaseballNumber
    * 1-9의 서로 다른 수로 이루어진 3자리의 수
2. 입력
    * 3자리의 수
3. 게임
    * 파일명 : Pitching
    * 스트라이크 : 같은 숫자, 같은 자리
    * 볼 : 같은 숫자, 다른 자리
    * 포볼/낫싱 : 같은 숫자가 없음.
4. 결과
    * 메소드 : Pitching.getPitchingResult
    * 게임의 결과가 3스트라이크일 경우 게임 종료. (PitchingResult.result == true)
5. 게임 종료 이후
    * 게임을 다시 시작하거나 완전히 종료.
